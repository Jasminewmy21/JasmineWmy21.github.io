1. git入门

准备工作

  1. 下载git
  2. 注册github
  3. 将git和github连接起来
    1)获取ssh密钥 id_rsa.pub
    2)在github的setting里找到ssh key, 新建ssh key并填写密钥

正式开始

  1. 确定git和github已经连接
    1)git init 初始化
    2)git remote add origin https://github.com/Jasminewmy21(用户名)/ProjectDSA(仓库名).git
    3)git pull origin master

  2. 本地文件推到github上
    1)git add Demo01.java(要上传的文件)
    1)git add . (文件夹下的全部文件)
    2)git comment -m ‘注释内容’ (可没有)
    3)git push origin master (推上去)
